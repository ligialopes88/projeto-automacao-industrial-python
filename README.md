# 🏭 Automação Industrial: Gestão de Qualidade e Armazenamento

Protótipo desenvolvido em Python para automatizar o controle de produção de peças industriais. O sistema realiza a triagem de qualidade automática, organiza o armazenamento em caixas e gera relatórios de eficiência. Projeto acadêmico para a disciplina de Algoritmos e Lógica de Programação - UniFECAF.

## 🚀 Funcionalidades
- [cite_start]**Cadastro de Peças:** Registro de ID, peso, cor e comprimento[cite: 9].
- [cite_start]**Validação Automática:** Verifica se a peça atende aos padrões (Peso: 95g-105g | Cor: Azul/Verde | Comprimento: 10cm-20cm)[cite: 10, 11, 13, 14].
- [cite_start]**Gestão de Estoque:** Armazenamento automático em caixas de 10 unidades[cite: 15, 16].
- [cite_start]**Relatórios:** Consolidação de aprovadas, reprovadas (com motivos) e caixas totais[cite: 17, 18, 20, 22].
